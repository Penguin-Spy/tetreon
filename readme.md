# Tetreon

TI-84+ CE Tetris clone with modern tetris rules (T-spins, SRS, Hold, etc.)

features:  
- [x] tetrimino rotation, movement, collision
- [x] proper tetrminio wallkicks/shifting when rotating
- [x] holding tetriminos
- [x] random tetriminos
